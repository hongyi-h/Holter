# 研究提案：基于大规模动态心电图的心脏数字表型发现

> 目标期刊：Nature / Nature Medicine
> 团队：临床（心内科）+ AI/CS
> 数据：10,000例 24h Holter（3通道, 128Hz, 逐拍标注）

---

## 1. 核心科学问题

**24小时连续心电图中是否存在尚未被识别的心脏动态表型，且这些表型具有独立的临床预后价值？**

现有Holter分析仅提取少量统计指标（平均心率、室早计数、HRV时域/频域参数），然后丢弃>99%的信息。我们假设：用深度学习直接从完整的24h心率动态序列中学习表征，可以发现传统分析遗漏的、具有临床意义的心脏动态模式亚型。

## 2. 为什么这能发Nature

| 要素 | 本研究如何满足 |
|------|--------------|
| **新发现** | 不是"用DL做分类"，而是"发现新的心脏表型" — Nature的核心叙事 |
| **先例** | Attia et al. (Nature Medicine, 2019) 证明"ECG里藏着你不知道的信息"这个故事Nature买单 |
| **独占优势** | 24h连续数据的时间结构（昼夜节律、活动-睡眠转换、心率恢复动态）是12导联静态ECG做不到的 |
| **临床转化** | 如果新表型能预测不良事件，直接改变Holter报告的临床解读方式 |
| **方法创新** | 首次在大规模Holter数据上做无监督表型发现（现有工作全部是监督分类） |

## 3. 文献定位

### 3.1 已占位的赛道（不要做）

- **心律失常自动分类**：Hannun 2019 (Nature Medicine), Ribeiro 2020 (Nature Comms) 已占位，且10k数据量无法与百万级数据集竞争
- **ECG基础模型预训练**：Oh 2024 (Nature Medicine), ECG-FM 2024 等已在静态ECG上做了，10k例做基础模型偏小
- **AI辅助Holter报告**：工程优化类工作，发不了Nature

### 3.2 我们的空白（没人做过）

- **无人在Holter长程数据上做过无监督表型发现**
- **无人用DL从24h RR间期序列的完整时间结构中提取表征**（现有HRV研究全部是手工统计特征：SDNN, RMSSD, pNN50等）
- **无人系统研究过心率昼夜节律模式的个体差异及其临床意义**

### 3.3 关键参考文献

| 论文 | 期刊/年份 | 与我们的关系 |
|------|----------|-------------|
| Hannun et al., Cardiologist-level arrhythmia detection | Nature Medicine, 2019 | DL+ECG范式奠基，但做短片段分类，非连续监测 |
| Attia et al., AI-ECG for low ejection fraction | Nature Medicine, 2019 | "ECG中发现隐匿信息"的叙事先驱 — 我们的精神前辈 |
| Ribeiro et al., Automatic 12-lead ECG diagnosis | Nature Comms, 2020 | 大规模多标签，但静态ECG |
| Raghunath et al., ECG prediction of mortality | Nature Medicine, 2020 | ECG→预后，但静态ECG |
| Oh et al., ECG foundation model | Nature Medicine, 2024 | ECG预训练里程碑，但仍是短程ECG |
| Biton et al., AF burden from Holter | European Heart Journal, 2021 | Holter+临床结局，但方法传统 |

## 4. 数据资产

### 4.1 现有数据

| 属性 | 值 |
|------|-----|
| 样本量 | 10,000例 |
| 波形 | 3通道 (含II导联, V5), 128Hz, uint8, ~32MB/例 |
| 时长 | ~24h/例 |
| 逐拍标注 | R点时间戳 + 类型 (N/V/F)，来自设备算法 |
| 临床摘要 | 年龄、性别、心率统计、室早/室上早计数、医生结论文本 |
| 总数据量 | ~320GB波形 + ~10万拍/例 × 10k例 = ~10亿个RR间期 |

### 4.2 数据优势

- **完整24h覆盖**：捕捉昼夜节律、睡眠-觉醒转换、活动响应
- **逐拍R点标注**：可直接提取RR间期序列，无需从头做R波检测
- **医生结论文本**：可用NLP提取结构化诊断标签作为弱监督信号

### 4.3 关键缺口（需确认）

- **结局数据**：能否通过PID回溯到医院EMR，获取后续诊断、住院、死亡等结局？
  - 如果有 → Nature Medicine级别（表型发现 + 预后验证）
  - 如果没有 → Nature Communications级别（表型发现 + 已知临床变量关联）
- **外部验证集**：是否有第二家医院的Holter数据可做外部验证？

## 5. 技术路线

### Phase 1：表征学习与表型发现（3-4个月）

**输入**：10k例 24h RR间期序列（每例~10万个点）

**方法**：
1. 将每例24h RR间期序列重采样为固定长度的心率动态曲线（如1440点 = 每分钟一个值）
2. 同时提取多尺度特征：
   - 分钟级：短时HRV纹理
   - 小时级：活动-休息模式
   - 全天级：昼夜节律形态
3. 用对比学习（如SimCLR/BYOL变体）或变分自编码器学习表征
4. 在学到的表征空间中聚类，发现心率动态模式亚型

**验证**：
- 聚类结果是否稳定（不同随机种子、不同子集）
- 亚型是否与已知临床变量（年龄、性别、室早负荷）有关联但不完全重合
- 如果完全重合 → 没有新发现，需要调整方法
- 如果完全无关 → 可能是噪声，需要检查

### Phase 2：临床意义验证（2-3个月）

**取决于结局数据是否可用：**

**路径A（有结局数据）**：
- 验证新发现的表型是否独立预测不良心血管事件
- Cox回归，校正年龄、性别、传统HRV指标后，新表型是否仍有独立预测价值
- 这是Nature Medicine的核心证据

**路径B（无结局数据）**：
- 验证新表型与医生结论文本中的诊断模式的关联
- 用NLP从结论文本中提取结构化标签（窦性心动过速/过缓、室早负荷高/低、ST-T改变等）
- 验证新表型是否能区分不同诊断组合，且提供超越传统指标的信息
- 这条路径上限是Nature Communications

### Phase 3：波形级深度分析（2-3个月）

- 在Phase 1发现有意义的表型后，回到原始波形数据
- 用注意力机制或梯度可视化，定位驱动表型差异的具体波形特征
- 目标：不仅发现"这些人不同"，还要解释"哪里不同" — 这是临床可解释性的关键

### Phase 4：论文撰写与外部验证（3-4个月）

- 如果有外部验证集：在独立数据上复现表型发现
- 撰写论文，核心叙事："我们从10,000例24小时连续心电图中发现了X种新的心脏动态表型，其中表型Y独立预测Z"

## 6. 风险与应对

| 风险 | 概率 | 应对 |
|------|------|------|
| 聚类结果仅反映年龄/性别，无新发现 | 中 | 在校正人口学变量后的残差上做聚类 |
| 设备标注（N/V/F）不准确 | 低-中 | 用原始波形训练R波检测器做质控 |
| 无结局数据，发现无法验证临床意义 | 取决于确认 | 退而求其次用诊断文本做弱验证，目标降为Nat Comms |
| 10k例不够发现稀有表型 | 低 | 10k对于主要表型足够；稀有亚型留给后续研究 |
| 128Hz采样率偏低，丢失高频信息 | 低 | 对RR间期分析无影响；波形分析时承认局限性 |

## 8. 时间线

| Phase 2: 临床验证 | Month 5-7 | 证明新表型的临床意义 |
| Phase 3: 波形解释 | Month 7-9 | 定位驱动表型差异的波形特征 |

## 9. 待团队讨论的关键决策

1. **结局数据**：能否通过PID回溯EMR？这决定了Nature Medicine vs Nature Communications
2. **外部验证**：是否有合作医院可提供独立Holter数据集？
3. **伦理审批**：回顾性研究的伦理审批流程和时间？
4. **数据质量**：10k例中有多少例记录完整（>20h）？设备标注质量如何？
5. **临床问题聚焦**：团队最关心哪类患者/哪种结局？（如：室早患者的猝死风险？房颤筛查？心衰早期识别？）
